Running the unit test

This will download all dependencies, compile the project, and run your project on a jetty instance. NOTE: THIS WILL TAKE QUITE SOME TIME THE FIRST RUN SINCE MAVEN WILL DOWNLOAD ALL THE DEPENDENCIES INCLUDING YOUR SERVELET CONTAINER (JETTY).

Run the test in the folder src/test/java in the usual way.

Starting Jetty

  • To run the maven commands, right-click on the project and choose Run As > Maven build...
  • In the Goals field, enter clean jetty:run

Running the Hello World Example

  • Start Jetty
  • Observe the wsdl is automatically automatically generated http://localhost:8080/message-cxf/MessageService?wsdl
  • View it working at http://localhost:8080/message-cxf/MessageService/getMessage?message=ThisMessageDoesntMatterSinceItIsHelloWorldImplementation You should see output similar as below: 
    <soap:Envelope>
 	<soap:Body>
		<ns1:getMessageResponse>
			<return>Hello World</return>
		</ns1:getMessageResponse>
	</soap:Body>
</soap:Envelope>

Running the Member Variable Example

You can switch the MessageService implementations by editing src/main/webapp/WEB-INF/applicationContext-cxf.xml.
  • Ensure you have stopped jetty (use the red button in the Maven build console)
  • Change the jaxws:endpoint's implementor attribute. It will change from <jaxws:endpoint id="messageService" implementor="#helloWorldMessageService" to <jaxws:endpoint id="messageService" implementor="#memberVariableMessageService"
  • Start Jetty
  • Observe the wsdl is still the same (it is based off the interface) and it is still automatically automatically generated http://localhost:8080/message-cxf/MessageService?wsdl
  • View it working at http://localhost:8080/message-cxf/MessageService/getMessage?message=ThisMessageDoesntMatterSinceItIsHelloWorldImplementation You should see output similar as below: 
    <soap:Envelope>
 	<soap:Body>
		<ns1:getMessageResponse>
			<return>The argument passed in is ThisMessageDoesntMatterSinceItIsHelloWorldImplementation.</return>
		</ns1:getMessageResponse>
	</soap:Body>
</soap:Envelope>
